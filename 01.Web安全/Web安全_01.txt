第一章
	渗透测试
		渗透测试通过模拟网络攻击来评估网络系统安全的一种方法。主要分为白盒测试和黑盒测试两种。

	渗透测试流程
		1、获得授权
		2、信息收集
				http://tool.chinaz.com/
			1、所用语言 php asp python …
			2、所用数据库  asp --> access; aspx --> sqlserver; php --> mysql; 等
			3、中间件  Apache …
			4、操作系统
		3、发现漏洞
		4、生成报告

	渗透测试应当注意的问题
		1、合法的前提
			合法约定
		2、合理的时间
			使用量较少的时间段
		3、合理的深度
			渗透不要过于深入。忌提权、后门
		4、尽量不影响网站运行
			手工测试，线程控制，忌产生大流量攻击
