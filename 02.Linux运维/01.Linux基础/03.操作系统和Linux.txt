操作系统

	操作系统就是沟通使用者和硬件之间传递信息的工具或程序。
	是计算机中负责支撑应用程序运行环境以及操作系统环境的基础系统软件，同时也是计算机系统的核心。

	常见的操作系统：
		windows
		Unix
		Linux

Unix操作系统
	
	Unix是一个强大的多用户、多任务操作系统。（Windows远程不支持多用户同时连接）
	Unix支持多种处理器架构。
		
	Unix具有技术成熟、可靠性高、网络和数据库功能强、伸缩性突出和开放性好（强大的开放功能）等特色。
	可以满足各行各业的需求，特别是企业重要业务的需求，已经成为主要的工作站平台和重要企业操作平台。
	Unix曾经是服务器操作系统的首选。但近年来与Windows Server以及Linux的竞争中略有失利。

	Unix曾大量的被运用于银行金融等行业。

Linux操作系统

	Linux 起源于 Minix 操作系统。早期用于操作系统的教学。
	Linux 是一套免费使用和自由传播的类Unix操作系统。
	
	Linux 继承了Unix几乎所有的优点。是由全世界各地成千上万的程序员共同设计实现的。
	其目的是建立不受任何商品化软件的版权制约的、全世界都能自由使用的Unix兼容产品。

	Linux 以它的高效和灵活著称。能够实现全部的Unix特性。具有多任务、多用户的能力。
	Linux 不仅包含完整的操作系统，而且还包含了文本编辑器、高级语言编辑器等应用软件。还包含多个窗口管理器的图形用户界面。

	Linux 是自由软件。可以无偿使用、无偿获取源码、无约束传播、自由修改。

Linux的常见版本

	CentOS、Ubuntu、Redhat
	CentOS 与 Redhat 的区别。
		CentOS可以理解为Redhat的一个再编译版本。
		是以Redhat所发布的源代码重建符合GPL协议的Linux。PS：Redhat9.0之后不再遵循GPL协议，为收费产品。

	以上三种版本都支持图形桌面。PS（图形桌面一般在安装Oracle等软件时会使用，其他情况很少使用图形桌面）
